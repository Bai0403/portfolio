04 — Automotive Industry Insights

汽车行业洞察 Cars · Brand · Campaign · Trends

Insight 01

汽车市场进入"存量竞争"时代

过去市场依靠新增需求驱动增长，而当前市场逐渐进入存量竞争阶段。面对销量增速放缓，运营工作的重点也从"获取更多用户"转向"提升用户价值"。相比追求短期增长，我认为更重要的是通过精细化运营提升用户留存、品牌认同及全生命周期价值。

Key Insight

存量时代，比获取用户更重要的是持续经营用户。

Insight 02

纯电化不仅是技术升级，更是用户体验升级

随着换电网络完善、电价下降及智能化能力持续提升，纯电车型的竞争优势已逐渐从成本转向综合体验。未来影响用户决策的不只是续航，而是补能效率、智能驾驶及智能座舱等整体体验。

Key Insight

用户购买的不只是纯电汽车，而是更高效、更智能的出行体验。

Insight 03

品牌竞争的核心是长期用户运营

随着产品配置逐渐趋同，品牌之间越来越难依靠硬件形成长期差异。真正建立竞争优势的，是品牌价值能否持续传递给用户。我认为总部制定品牌策略只是起点，更重要的是通过区域传播、一线销售及用户运营，将品牌理念转化为真实的用户体验。

Key Insight

品牌价值最终不是靠传播建立，而是靠用户体验不断验证。

06 — Game UX Research

Game UX Research

Game UX

第五人格（Identity V）

Observation

《第五人格》通过 1V4 非对称竞技 与无局内语音设计，提高了团队协作门槛，也让获胜后的成就感更强。持续推出联动内容及创新玩法（如模仿者、加页手记），不断延长产品生命周期。随着用户增长，社区也面临玩家对立、饭圈化等问题，但活跃的玩家文化和二创内容依然是产品持续出圈的重要原因。

Key Insight

玩家社区创造的文化价值，往往比官方内容更能推动产品破圈。

Game UX

王者荣耀（Honor of Kings）

Observation

《王者荣耀》以成熟的 5V5 MOBA 玩法构建稳定竞技体验，个人能力、资源运营与团队配合共同影响对局结果，高水平玩家能够带动团队获胜。通过持续版本更新、赛事运营及较低门槛的皮肤商业化，保持用户活跃与长期留存。

Key Insight

成熟玩法保证留存，持续运营与商业化推动产品长期增长。
